Fast and affordable way to build highly customizable exit survey forms for Wordpress websites

== Description ==

This plugin adds the required javascript of your Mayakho project to your Wordpress website.

For more information visit:

[Mayankho](http://www.mayankho.com?utm_medium=partners&utm_source=Wordpress&utm_campaign=plugin-directory&utm_content=description)

== Installation ==

1. Install the Mayankho plugin through your Wordpress admin or upload the files to your plugin folder
2. Get the Mayankho code from your Mayankho project interface
3. Paste the Mayankho code into your Mayankho plugin in your Wordpress admin
4. Press save to enable the Mayankho project to work
